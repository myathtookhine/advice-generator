# Advice App

Click button. Get advice. Try not to mess it up.

**Features**
- **Random Advice:** Because you clearly don't know what you're doing.
- **Translation:** Burmese support via Gemini AI.
- **Daily Quotes:** One fake-deep thought per day.
- **Brutalist UI:** No gradients. No shadows. Just raw disappointment.

**How to run**
1. Clone this dump.
2. `npm i` then `npm run dev`.
3. Add `VITE_GEMINI_API_KEY` in `.env`. Or don't, and watch it fail.
